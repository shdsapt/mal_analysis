- 해당 분석은 이미지 파일 (.png, .jpeg 등) 에 대한 파일 분석 절차로, 아래 순서에 따라 수행해줘

1. 내가 요청한 파일에 대해 file 명령어 실행하고 그 결과를 출력 및 해석 
   - ex) file suspicious.png
2. exiftool 명령어를 실행하여 요청 파일의 메타데이터를 분석하고 결과 출력
   - ex) exiftool suspicious.png
3. 마지막으로 strings 명령어를 실행하여 악의적인 링크 및 파일 포함 여부를 확인해줘
   - ex) strings suspicious.png | grep -Ei 'html|script|zip|pk..|<svg'
